Mi primer Layout

En este ejercicio aprendí a usar una cuadricula imaginaria llamada "Layout" la cual sirve para distribuir los elementos de texto, imágenes, etc. en mi página web. También puse en práctica el ejercicio anterior que consistía en insertar texto encima de una imagen respetando la cuadricula.

Cinema XXI

En este ejercicio puse en práctica mis conocimientos acerca de la grid system en cada una de las secciones de mi página web, así como también conocí etiquetas nuevas para insertar un video de YouTube con la etiqueta iframe y agregar una barra de buscador con la etiqueta input.

GRID SYSTEM

Considero que este ejercicio fue el más difícil pero el que mejor me sirvió para poder entender con claridad para que servía la grid system y cómo podía ocuparla en todos los casos.

MI PERFIL

En este ejercicio puse en práctica todo lo aprendido durante 3 semanas, ya que el objetivo era incluir en una página web información acerca de ti y cosas que te gusten.
